## 特征设计

### 基础特征（11个）
1. **8个匹配方式标志位**：从文件名提取（0=Exact, 1=Ternary, 2=Wildcard）
2. **normalized_size**：归一化的表大小
3. **exact_count**：Exact 匹配字段数量
4. **wildcard_count**：Wildcard 匹配字段数量

### 派生特征（8个）
1. **match_complexity**：`exact_count + wildcard_count × 2`
2. **wildcard_ratio**：`wildcard_count / (exact_count + wildcard_count)`
3. **size_complexity_interaction**：`normalized_size × match_complexity`
4. **match_type_complexity**：基于匹配方式的复杂度分数
5. **exact_match_count**：Exact 匹配类型数量
6. **ternary_match_count**：Ternary 匹配类型数量
7. **wildcard_match_count**：Wildcard 匹配类型数量
8. **wildcard_match_ratio**：Wildcard 匹配类型比例

## 目标变量

**table_memory_estimate_bytes**：表内存估计值（字节）

## GPU vs CPU 训练分析

### 推荐：使用 CPU 训练

**理由**：
1. **数据规模适中**：8000 条记录，CPU 处理速度已足够
2. **算法特性**：随机森林、梯度提升等树模型主要依赖 CPU
3. **scikit-learn**：默认不支持 GPU，使用 `n_jobs=-1` 可充分利用多核 CPU
4. **Windows 环境**：避免 GPU 配置复杂性
5. **预计训练时间**：
   - 基础版：30-60 秒
   - V2 版：2-5 分钟

### GPU 训练场景（不推荐）
- 数据规模 > 100万条
- 需要使用深度学习模型
- 有专门的 GPU 加速库（如 XGBoost GPU 版本）

## 预期性能指标

基于 Tofino 框架经验和数据规模，预期：

- **R² Score**：> 0.85（优秀）
- **MAE**：< 5000 bytes（相对误差 < 5%）
- **RMSE**：< 10000 bytes
- **MAPE**：< 10%（平均绝对百分比误差）
- **过拟合控制**：训练 R² - 测试 R² < 0.15
